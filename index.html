Example of FancyBox lightbox

Steps to create a webpage with this viewer:
  1. Add jQuery and FancyBox scripts to the header of your webpage: 
    <!-- Below we include the jQuery Library -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

<!-- Add fancyBox -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/fancybox/3.5.7/jquery.fancybox.min.css" type="text/css" media="screen" />
<script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/fancybox/3.5.7/jquery.fancybox.min.js"></script>

  2. Create a set of linked images:

    <div class="gallery">
<a href="fullsize.png"><img src="thumbnail.png"></a>
<a href="fullsize.png"><img src="thumbnail.png"></a>
<a href="fullsize.png"><img src="thumbnail.png"></a>
...
</div>

    For better performance, use thumbnails in the img src.

  3. At the end of the page, add the script to start FancyBox:

    <script>
$(document).ready(function() {
  $(".gallery a").fancybox();
});
</script>

Detailed instructions, options, examples and tips & tricks can be found on the FancyBox site.
